# A股宽基ETF投资组合 VaR 蒙特卡洛风险模拟
基于正态分布 & 对称学生t分布蒙特卡洛模拟的投资组合尾部市场风险度量

## 项目简介
本项目针对A股宽基指数ETF投资组合，采用**历史模拟法、正态分布蒙特卡洛、对称学生t分布蒙特卡洛**三种主流金融风险模型，测算95%、99%置信水平下组合VaR风险价值。

对比不同分布假设对A股收益率风险测算的影响，实证探究正态分布、对称厚尾t分布在A股市场的真实适配效果与模型固有缺陷。

## 数据选取
- 标的资产：沪深300ETF、中证500ETF、创业板ETF、科创50ETF
- 组合方式：四只ETF等权重构建分散化投资组合
- 数据类型：日度收盘价 → 计算对数日收益率序列
- 核心研究指标：组合收益率均值、波动率、VaR尾部风险价值

## 研究方法
1. **历史模拟法**
以历史真实损益分布作为风险基准，测算A股市场真实尾部下行风险，作为所有模型对比标杆

2. **正态分布蒙特卡洛模拟**
假设资产收益服从标准正态分布，通过大量随机情景模拟未来损益分布，分位数计算组合VaR

3. **对称学生t分布蒙特卡洛模拟**
通过最大似然估计MLE拟合历史数据得到最优自由度df，利用金融经典尖峰厚尾t分布开展蒙特卡洛风险模拟

## 实证结果
| 测算方法 | 95% VaR | 99% VaR |
| ---- | ---- | ---- |
| 历史模拟法 | 2.02% | 3.24% |
| 正态分布蒙特卡洛 | 2.31% | 3.31% |
| 对称t分布蒙特卡洛 | 3.03% | 5.18% |

## 核心结论
1. A股宽基ETF收益率显著不符合标准正态分布，具备典型**尖峰厚尾、收益左偏**市场特征
2. 正态分布本次数值贴合历史风险，但理论天生低估极端黑天鹅尾部崩盘风险，结果仅为样本巧合
3. 对称学生t分布左右尾部完全对称，无法匹配A股「慢涨快跌、暴跌远多于暴涨」的非对称行情
4. 对称t分布无论如何调整自由度，均会过度高估极端下行风险，A股实战测算效果劣于传统正态分布 

## 项目不足与未来优化方向
1. 仅使用对称型t分布，未采用更贴合A股行情的偏态t分布模型
2. 未考虑A股涨跌停制度、流动性约束等本土市场特殊规则
3. 后续可使用Cornish-Fisher高阶矩修正正态分布，大幅优化尾部风险拟合精度
